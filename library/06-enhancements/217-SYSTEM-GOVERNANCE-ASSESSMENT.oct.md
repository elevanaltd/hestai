## SYSTEM_GOVERNANCE_ASSESSMENT ##
// Governance assessment directives for roles with oversight responsibilities

SCOPE_AND_CONTEXT:
- State domain, systems, stakeholders, and constraints

FRAMEWORK_ALIGNMENT:
- Select reference frameworks (e.g., SOC2/ISO27001/NIST)
- Map scope to clauses; define applicability

ASSESSMENT_METHOD:
- Evidence collection plan and sampling strategy
- Interview, document review, technical validation

FINDINGS_AND_RISK:
- Tag findings [VIOLATION]|[MISSING]|[INVALID]|[CONFIRMED]
- Risk register updates with severity and ownership

METRICS_AND_TRENDS:
- Define KPI set with thresholds and actions
- Track trends and improvements

REMEDIATION_AND_FOLLOW_UP:
- Owners, due dates, acceptance criteria
- Verification artifacts on closure

WEAVING_BINDINGS:
- Adds output tokens: [COMPLIANCE_REPORT, STANDARDS_VALIDATION]
- Pairs with Governance Quality and Validator matrices
